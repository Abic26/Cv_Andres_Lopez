Habilidales & Proyectos

HTML

Como Front-End Junior, creo estructuras web con HTML y aplico estilos visuales con CSS. Mi objetivo es hacer que el sitio web sea atractivo, funcional y fácil de usar para los usuarios, colaborando con diseñadores y mejorando la experiencia con interacciones en JavaScript.

JAVASCRIPT

Además, me encargo de la validación de formularios en el lado del cliente, asegurándome de que los datos ingresados por los usuarios cumplan con los requisitos especificados. Utilizo JavaScript para verificar y validar los campos de entrada antes de enviar los datos al servidor, ademas de poder almacenar informacion en base de datos locales y externas, consultandolas por medio de metodos de JS

CSS

Como Front-End Junior, aplico CSS para estilizar y mejorar el aspecto visual de las páginas web. Trabajo junto a diseñadores y equipos de UX/UI para implementar diseños en código CSS, asegurando coherencia visual, legibilidad y accesibilidad en el sitio web.

Pagina Web

Mi proyecto de pagina web aplicada a una pyme real de colombia, la cual les ayuda a los clientes interactuar por medio de uas funciones que se le dieron a la pagina por medio de JavaScript, como enviar mensaje por whatsapp o ir a la ubicacion de la Boutique, se le dio estilos por medio de css a gusto de owner, me base en un mockup que el owner ya tenia previamente y le dimos caña para poderlo implementar en codigo.

Calcular Creditos

Mi sistema de creditos es una interfaz la cual se apoya de JavaScript para calcular creditos que se vayan a solicitar un cliente-usuario, se utilizo html para maquetar la estructura y css para darle los estilos a la interfaz y sea mas dinamica para los usuarios.

Sistema de bodegaje

Mi sistema de bodegajes el cual se puede aplicar a cualquier tipo de empresa, en este ejempo lo realice a una empresa que maneja una bodega de arroz, el cual se realizo por medio de html, css y fundamentalmente JavaScript para darle funcionalidad a las acciones que tiene esta aplicacion.
